プライバシーポリシー

最終更新日：2026年4月10日

Cookey（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の適切な取り扱いに努めます。本ポリシーは、本アプリが収集・使用する情報について説明します。

1. 収集する情報

・食材・献立データ
ユーザーが入力した食材リスト、生成された献立、お気に入りレシピ、買い物リスト、食材在庫などの情報。
・健康情報
体重・身長・年齢・性別・目標体重など、ユーザーが任意で入力するプロフィール情報。
・広告識別子
Google AdMob による広告配信のため、端末の広告識別子（IDFA / GAID）が使用される場合があります。
・アプリの使用状況
アプリのクラッシュ情報や機能の利用状況など、改善目的のデータ。個人を特定しない形で収集されます。

2. 情報の使用目的

・献立提案機能の提供
入力された食材・健康情報をもとに、AI による献立提案・栄養計算などの機能を提供するために使用します。
・広告の表示
Google AdMob を通じて、ユーザーに関連性の高い広告を表示するために使用します。
・アプリの改善
使用状況データをもとに、機能の改善・不具合修正を行います。

3. 第三者への提供

本アプリは以下のサービスを利用しており、各サービスの個人情報取り扱いポリシーが適用されます。

・Google AdMob
広告配信
https://policies.google.com/privacy
・Google Gemini API
AI 献立提案機能（入力された食材等のプロンプトが送信されます）
https://policies.google.com/privacy
・Oisix（オイシックス）
アフィリエイトリンク（食材購入先のご案内）
https://www.oisix.com

4. データの保存

🔒 食材・献立・健康情報などのすべてのデータは、端末内にのみ保存されます。外部サーバーへの送信は行いません。

ただし、AI 献立提案機能（献立生成・食材認識など）を利用する際は、入力内容（食材リスト・条件など）が Anthropic の API サーバーに送信されます。送信されるデータに氏名・連絡先などの個人情報は含まれません。

アプリを削除した場合、端末内に保存されたすべてのデータは消去されます。

5. 広告について

本アプリは Google AdMob による広告を表示しています。広告のパーソナライズを希望しない場合は、端末の設定から「広告のパーソナライズを無効にする」または「広告 ID をリセット」することができます。

また、アフィリエイトリンク（Oisix）を含む場合があります。該当箇所には「※PR」または「広告」の表記を行っています。

6. お子様のプライバシー

本アプリは 13 歳未満のお子様を対象としていません。13 歳未満の方からの個人情報は意図的に収集しません。

7. プライバシーポリシーの変更

本ポリシーは必要に応じて更新される場合があります。変更があった場合は、本ページの「最終更新日」を更新します。継続してアプリをご利用いただくことで、改訂後のポリシーに同意したものとみなします。
